# 🌾 Indian Agriculture Crop Production Predictor

A beginner-friendly Machine Learning project that predicts agricultural crop production in India based on crop type, state, season, and cost of cultivation. 

## 🎯 Project Goal
The goal of this project is to provide a simple, AI-powered tool for farmers, researchers, and students to estimate crop output. By analyzing historical trends, the model helps in understanding how much production can be expected given specific costs and environment variables.

## ✨ Features
- **Machine Learning**: Uses a **Linear Regression** model for continuous output prediction.
- **Interactive UI**: A modern web interface built with **Streamlit** (Python).
- **Responsive Design**: Includes a premium "Glassmorphism" look with agricultural themes.
- **Cloud Deployment**: Hosted on **Streamlit Community Cloud** via GitHub.

## 🛠️ Technology Stack
- **Languages**: Python 3.x
- **Data Handling**: Pandas, NumPy
- **Machine Learning**: Scikit-Learn
- **Web App**: Streamlit, CSS
- **Model Storage**: Pickle

## 📂 Project Structure
- `app.py`: The main Streamlit web application.
- `train_and_save.py`: Script to train the ML model and save it as a package.
- `crop_model_package.pkl`: The serialized trained model and label encoders.
- `requirements.txt`: List of Python libraries required for the project.
- `static/`: Folder containing background images and CSS assets.
